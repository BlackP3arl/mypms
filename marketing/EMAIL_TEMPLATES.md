## Email Best Practices

1. **Subject Line:**
   - Keep under 50 characters
   - Use emojis sparingly (1 max)
   - Create curiosity or provide value
   - A/B test different approaches

2. **Preview Text:**
   - Complement subject line
   - 40-90 characters
   - Don't repeat subject line

3. **Content:**
   - Get to the point quickly
   - Use short paragraphs (2-3 lines)
   - Include visual breaks (â”â”â”)
   - One clear CTA per email

4. **Timing:**
   - Best days: Tuesday, Wednesday, Thursday
   - Best times: 10am, 2pm, 8pm (user's timezone)
   - Avoid Mondays and weekends

5. **Personalization:**
   - Use first name when possible
   - Segment by user behavior
   - Reference specific actions they took

6. **Mobile Optimization:**
   - 60% of emails opened on mobile
   - Large CTAs (48px+ tap targets)
   - Single column layout
   - Readable font sizes (16px+)

- **Conversion Rate**: Target 2-10%
